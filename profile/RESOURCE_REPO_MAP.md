## How to use this table

- **Main Credit Plainly guide** — the primary on-site deep link for that repo’s audience. Additional links belong inside each repository’s README and `data/` maps, not duplicated here.
- **Official source category** — representative government or program sources cited in the repo; see each repo for full URL lists.
- **Maintenance status** — `Active` means the repository is public and intended for ongoing link and content checks.

## Suggested update cadence

| Check | Frequency |
|-------|-----------|
| Broken official `.gov` links | Quarterly or when a reader reports an issue |
| Credit Plainly guide URLs in repo READMEs | When a related site guide is materially updated |
| New public repo row in this table | When a new educational repository is published under the org |
